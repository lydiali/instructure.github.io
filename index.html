  • Learn Canvas

    If you’re running Canvas yourself or considering working on the Canvas codebase, start here.

  • Learn API
    A good place to start learning is to work through the module on APIs in the Canvas Dev course. Still have questions? Join the mailing list or ask them in our IRC channel.
  • API Docs
    Canvas has API docs. If you’re new to them or to working with APIs in general you’ll probably want to start with the getting started page or review the API use cases.
  • Dev Key Signup
    Before you can use the Canvas API in a production environment you’re going to need a developer key. It’s easy to request a key, just reach out to us with your name, email, and what you’re hoping to do with the API.
  • Learn LTI
    There’s a module on LTI in the Canvas Dev course, definitely give it a looksie if you’re building your first LTI app. Still have questions? Join the mailing list or ask them in our IRC channel
  • LTI Docs
    There are official LTI docs, but if you’re struggling sort through them, we have a shorter LTI introduction you can check out to get started. We’ve collected some blog posts on LTI that you may find useful as well.
  • Submit an App
    Once you’ve built an LTI app and you’re ready to share it with the world, you can submit it to our app directory. You can also use that page to suggest apps you think somebody else should build (maybe, someday).
  • Mobile App API
    Our mobile apps on Android and iOS have an API that allows other apps on a device to communicate with them. If you have an app that would benefit from interacting with our Canvas Mobile apps, check out our App API documentation.
  • Learn Canvas
    If you’re running Canvas yourself or considering working on the Canvas codebase, first check out the introduction on GitHub. You should also join the mailing list or our IRC channel. Issues? Visit our help portal.
  • GitHub
    Canvas is released open source under the AGPL license. There’s a couple other related libraries you can see on our github site, or check out third-party codebases related to Canvas.
  • Contribute
    Canvas is dual-licensed, so if you’re interested in contributing to the project you’re going to need to send us a code contributor agreement. You’ll also probably want to review the contribution guide. After that it’s pull requests ftw!
